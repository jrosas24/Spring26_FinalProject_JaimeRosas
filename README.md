# Spring 2026 Final Project – NHANES Public Health Data Analysis

## Overview
This project analyzes NHANES public health data using R Studio. The project examines relationships between BMI, physical activity, sex, education, height, and weight among U.S. adults.

## Research Question
Is there a significant difference in BMI across physical activity levels among adults in the United States?

## Dataset
NHANES Public Health Data:
- Demographics
- Body Measurements
- Physical Activity Questionnaire

## Tools Used
- R Studio
- tidyverse
- ggplot2
- dplyr
- psych
- GitHub

## Project Structure
- `data/` → cleaned datasets
- `scripts/` → all R scripts and reports
- `output/` → graphs and statistical outputs
- `codebook/` → variable documentation
